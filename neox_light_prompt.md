An ethereal, airy light mode dashboard for Neox Command Center, featuring the "Antigravity" glassmorphic aesthetic over a cloud-based pearl background.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Theme: Light Mode, Ethereal, Premium "Antigravity" style
- Background: Pearl Ethereal (#f8fafc) with a soft radial pearl-to-white gradient (#ffffff).
- Surfaces: Frosted glass panels (#ffffff40) with a subtle backdrop-blur (16px) and a thin 1px light gray "Silk Border" (#00000008).
- Primary Accent: Deep Velocity Blue (#0284c7) for active navigation and high-contrast labels.
- Secondary Accent: Vibrant Amethyst (#9333ea) for financial growth visuals.
- Positive Action: Natural Moss (#16a34a) for safety (HSE) success.
- Warning/Alert: Sunset Ember (#ea580c) for critical logistics bottlenecks.
- Typography: "Space Grotesk" for headers, "JetBrains Mono" for technical data values.
- Effects: Soft, whisper-diffused shadows for elevation, micro-animations on interactive glass surfaces.

**Page Structure:**
1. **Glass Sidebar:** Slim, pill-shaped navigation container on the left with high-contrast icons for Dashboard, Finance, SCM, HRM, HSE, and Reports.
2. **Global Header:** "NEOX COMMAND CENTER" in bold Space Grotesk, with an airy user profile widget.
3. **Hero Analytics Grid:** Large, glowing Amethyst area chart for Finance, a "Pulse" list for SCM with Sunset alerts, a Moss-green safety gauge for HSE.
4. **Quick Actions Float:** Floating action buttons in Deep Velocity Blue with soft glowing elevations.
5. **Footer:** "Powered by Neox" with a live system status indicator.
